Registers

A: Accumulator

Z: Anti Accumulator (-A)

S: Stack, gives 0 if empty.

I: Next available input in numerical form, gives 0 if empty

Commands

a, acc[n/Z/S/I]: Add a number or register to A

b, jgz[n/Z/S/I/tag]: Jump a number or register amount or tag of instructions if A > 0

c, jez[n/Z/S/I/tag]: Jump a number or register amount or tag of instructions if A = 0

d, jlz[n/Z/S/I/tag]: Jump a number or register amount or tag of instructions if A < 0

e, jmp[n/Z/S/I/tag]: Jump a number or register amount or tag of instructions

g, giv[n/Z/S/I/tag]: Assign a number or register to A

o, oc: Output A as an ascii encoded character

n, on: Output A as a number

p, push: Push A onto S

l, pop: Pop S onto A

f, flush: Remove all pushed values from S

z, nop: Do nothing

*: Attaches to the instruction behind it. After the instruction executes, the running environment will breakpoint.

:tagname: : Put colons around a name to make it a tag. I can be used as an argument in jump instructions. The instruction will jump to the location of that tag

#comment#: Put hashtags around a comment. It will be ignored by the interpreter.
